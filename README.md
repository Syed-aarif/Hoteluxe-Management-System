# 🏨 Hoteluxe Management System

A Hotel Management System backend application developed using **Spring Boot**, **Spring Data JPA**, and **MySQL**.

---

## 📌 Features

* 👤 Customer Management (CRUD)
* 🛏️ Room Management (CRUD)
* 📅 Booking Management (CRUD)
* ✅ RESTful APIs
* ✅ Exception Handling
* ✅ Input Validation
* ✅ MySQL Database Integration

---

## 🛠 Technologies Used

* Java 17
* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL
* Gradle
* Postman
* Git
* GitHub

---

## 📂 Modules

* Customer Module
* Room Module
* Booking Module

---

## 🚀 API Endpoints

### Customer APIs

* POST `/customers/add`
* GET `/customers/all`
* GET `/customers/{id}`
* PUT `/customers/update/{id}`
* DELETE `/customers/delete/{id}`

### Room APIs

* POST `/rooms/add`
* GET `/rooms/all`
* GET `/rooms/{id}`
* PUT `/rooms/update/{id}`
* DELETE `/rooms/delete/{id}`

### Booking APIs

* POST `/bookings/add`
* GET `/bookings/all`
* GET `/bookings/{id}`
* PUT `/bookings/update/{id}`
* DELETE `/bookings/delete/{id}`
